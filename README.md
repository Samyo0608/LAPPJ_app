# L-APPJ 雷射輔助大氣噴流電漿製程應用程式介面

### 前端: React.js + Tailwind + Flowbite
### 後端: Python + Flask
### SQL: SQLite
### 整合: Electron.js
### 遠端同步: Socket.io
### 設備連線方式: Modbus RTU -> RS-232 or RS-485
### 設備連線方式: RK-512
### 設備連線方式: Ethernet TCP/IP
